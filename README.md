# Resurectum

> Resurect process for old (or new) frontend project

## How to use it

```bash
$ curl -L https://raw.githubusercontent.com/antistatique/resurectumum/dist/[mac OR win OR linux] > resurectum && chmod 755 ./resurectum && ./resurectum
```

## Contribution

### Install and build

```
$ yarn
$ yarn build
```

### Re-deploy build

```
$ yarn deploy
```
